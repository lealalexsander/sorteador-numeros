# Sorteador de Números

## Descrição
Este projeto é um sorteador de números desenvolvido com HTML, CSS e JavaScript. Ele permite ao usuário sortear uma quantidade específica de números dentro de um intervalo definido. Foi criado por Alexsander Leal como parte do curso de front-end da Alura, com foco em JavaScript. Atualmente estou no dia 40 do desafio #100DaysOfCode.

## Funcionalidades
- Sorteio de uma quantidade específica de números dentro de um intervalo definido pelo usuário.
- Evita a repetição de números sorteados.
- Interface amigável e responsiva.

## Como usar
1. Insira a quantidade de números a serem sorteados.
2. Defina o intervalo de números (de x até y).
3. Clique no botão "Sortear".
4. Os números sorteados serão exibidos na tela.
5. Para reiniciar o sorteio, clique no botão "Reiniciar".

## Estrutura do Projeto
O projeto é composto pelos seguintes arquivos principais:

- `index.html`: Contém a estrutura HTML da página.
- `style.css`: Contém o estilo CSS para a página.
- `app.js`: Contém o script JavaScript que realiza o sorteio dos números.

## Como Funciona
### JavaScript
O script JavaScript é responsável pela lógica do sorteio dos números. Ele obtém os valores de entrada do usuário, gera números aleatórios dentro do intervalo especificado e os exibe na tela, garantindo que não haja repetição de números sorteados.

### CSS
O estilo CSS garante que a interface seja visualmente atraente e responsiva. Utiliza-se um gradiente de fundo, elementos centralizados e uma tipografia clara para melhorar a experiência do usuário.
